git reset

  • -–hard 强制回退版本，不保留工作区和暂存区的修改。

  • –soft 回退到某一版本，保留工作区和暂存区的修改。

  • –-mixed（默认） 回退到某一版本，保留暂存区的修改，不保留工作区的修改。
